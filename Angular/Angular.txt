2.0 # CORE CONCEPTS 

    2.1 # BUILDING BLOCKS 

        ** Angular building blocks are:

            - Components 
            - Routes 
            - Directives 
            - Services 
            - Modules



    2.2 # COMPONENTS

        Everything in angular is a component.
        A component uses the notation component_name.component.ts .
        A component os composed by :

            - Component class 
            - templatUrl ( the url to the html the component uses)
            - styleUrl ( the url to css the component uses)

        example : 

            import { Component } from '@angular/core';

            @Component({
            selector: 'app-root',
            templateUrl: './app.component.html',
            styleUrls: ['./app.component.css']
            })
            export class AppComponent {
            title = 'my-app';
            }

        
        Component 
            -------> Component class
            -------> HTML 
            -------> CSS 

    2.3 # SERVICES
